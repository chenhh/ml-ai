# 共軛分佈(conjugate distribution)

## 簡介

| 先驗分佈(prior) | 資料分佈(data)  | 後驗分佈(posterior) |
| ----------- | ----------- | --------------- |
| Beta        | Binomial    | Beta            |
| Gamma       | Poisson     | Gamma           |
| Normal      | Normal      | Normal          |
| Dicichlet   | Multinomial | Dirichlet       |
