# 保凸運算

## 簡介

可用凸集構造出其它凸集。常用保凸運算如下：

* 交集
* 仿射函數
* 透射函數(perspective function)
* 線性分式函數(linear-fractional function)

## 交集(intersection)

> * 若$$S_1, S_2$$為凸集合，則$$S_1 \cap S_2$$也是凸集合。
> * 若$$\{ S_i\}, ~i\in \mathbb{N}$$為凸集合，則$$\cap_{i\in \mathbb{N}}S_i$$為凸集合。(註空集合為凸集合)。
