# 有限計算能力下的可排除集

## 摘要

我們研究了在具有向量收益的重複賽局中，當其中一名玩家被限制使用具有有限計算能力的策略時，可排除性的概念。 我們證明，當玩家 1 被限制僅使用有界回憶策略時，玩家 2 可以排除一個不包含凸可接近集合的閉集合。我們還證明，當玩家 1 被限制使用可以通過有限自動機實現的策略時，玩家 2 可以將他從任何的非凸閉集合中排除出去，該集合的凸包是最小凸可接近的。
